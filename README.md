# HCMV Multi-Epitope Vaccine Design

## Overview
This project focuses on the computational design of a multi-epitope vaccine candidate against Human Cytomegalovirus (HCMV AD169 strain) using immunoinformatics and structural bioinformatics approaches.

The study integrates epitope prediction, vaccine construct engineering, structural modeling, and molecular docking to evaluate the interaction between the designed vaccine construct and immune receptor TLR-4.

---

## Research Workflow
- Retrieval of HCMV proteome
- Antigenicity and immunogenicity screening
- CTL, HTL, and B-cell epitope prediction
- Vaccine construct design using linkers and adjuvants
- Physicochemical property analysis
- 3D structural modeling
- Structure validation
- Molecular docking with TLR-4 receptor
